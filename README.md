# Portfolio Website

This is a simple **responsive website** where all sections are aligned in a **single horizontal line**. The design ensures smooth scrolling while maintaining a modern UI.

## 🚀 Features
- **Fully Responsive** – Works on all devices.
- **Horizontal Alignment** – All sections are placed in a single row.
- **Smooth Scrolling** – Enables horizontal scrolling for overflow content.
- **Modern UI** – Clean design with a structured layout.

## 🛠️ Technologies Used
- **HTML5** – Markup structure.
- **CSS3** – Styling and layout.
